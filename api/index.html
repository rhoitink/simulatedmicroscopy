    __init__(image, pixel_sizes=None, metadata=None)

    Initialize an image, used as a wrapper to hold both an N-dimensional image and its pixel sizes

    Parameters:

    Name Type Description Default
    image ndarray

    2D or 3D numpy array containing pixel values, order should be (z,)y,x

    		 required
    pixel_sizes list[float]

    List/array of pixel sizes in meters, same length as number of image dimensions. Order: (z,)y,x. By default 1.0 m/px for all dimensions.

    		 None
    metadata dict

    Place to store custom metadata about an image, will also be stored in .h5 file.

    		 None

    add_read_noise(SNR=50.0, background=0.0)

    Add read noise to the image, realised by adding noise sampled from a normal distribution. The standard deviation of the distribution is constant across the entire image and determined by the SNR parameter, it is therefore signal-independent.

    Parameters:

    Name Type Description Default
    SNR float

    The signal-to-noise ratio of the resulting image, by default 50.0

    		 50.0
    background float

    Optional offset of the background, this will be the mean of the normal distribution, by default 0.0

    		 0.0

    Returns:

    Type Description
    type[Image]

    The image with read noise

    add_shot_noise(SNR=30.0)

    Add shot noise to the image, realised by sampling the intensities of the image from a Poisson distribution. At each pixel, the current value (after rescaling) dictates the mean value of the Poisson distribution, yielding signal-dependent noise.

    Parameters:

    Name Type Description Default
    SNR float

    The signal-to-noise ratio of the resulting image, by default 30.0.

    		 30.0

    Returns:

    Type Description
    type[Image]

    The image with shot noise

    convolve(other)

    Type Description
